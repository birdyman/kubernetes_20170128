
====================================================
Part 3: HPA Configuration:
====================================================
0. Enable Heapster:
minikube stop

minikube start --extra-config kubelet.EnableCustomMetrics=true

3. Enable heapster
minikube addons enable heapster
minikube addons open heapster
minikube addons list

1. Create Namespace by command:
	kubectl create -f webtest_deploy_hpa.yml

2. Apply HPA by command:
	kubectl autoscale deployment/webtest --min=1 --max=10 --cpu-percent=10 

3. Monitor by command:
	kubectl get hpa
	kubectl describe hpa/<hpa name>

4. Check Heapster:
	kubectl get heap
	kubectl get pods --namespace=kube-system
	kubectl logs -f heapster-4nfdm -n kube-system

5. Generate Load:
	kubectl run -i --tty load-generator --image=busybox /bin/sh
	while true; do wget -q -O- http://php-apache.default.svc.cluster.local; done

	kubectl exec <pod name> -c webtest md5sum /dev/urandom
